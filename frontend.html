Construa Websites Responsivos, modernos e poderosos.

De forma direta, o front-end é a área do desenvolvimento voltada, principalmente, para as questões visuais e de usabilidade.

3 Tecnologias Principais

HTML, CSS E JAVASCRIPT

Essas são as três principais linguagens utilizadas no front-end da nossa aplicação, ou seja, no lado do cliente no próprio navegador.

HTML

A estruturação do website.

HTML é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores. O HTML é o responsável pela estruturação do website.

Aprender HTML

CSS

Dando estilo ao seu website.

Cascading Style Sheets é um mecanismo para adicionar estilo a um documento web (HTML). Pode ser aplicado ao website por meio de um arquivo separado ou inserido diretamente no código HTML.

Aprender CSS

JavaScript

A interatividade do seu website.

JavaScript é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma. Juntamente com HTML e CSS, o JavaScript é uma das três principais tecnologias da World Wide Web.

Aprender JavaScript
